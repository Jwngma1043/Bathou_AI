# 🌿 Bathou AI - Bodo Language Chatbot

Bathou AI is a regional language chatbot developed to support communication and learning in the Bodo language. The chatbot uses a custom Bodo language dataset and fuzzy string matching to understand user queries and provide responses in Bodo along with English translations.

## 📌 Project Overview

Bathou AI is designed to promote the Bodo language through an interactive chatbot that can answer greetings, questions, cultural queries, and number-related inputs.

This project was developed as a mini-project using React, Node.js, Express, and a custom Bodo language dataset.

## 🚀 Features

- Interactive chatbot interface
- Bodo language support
- English translations
- Fuzzy matching for user input
- Bodo culture information
- Number translation support
- Real-time responses

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Jwngma1043/Bathou_AI.git
cd Bathou_AI
```

### Install Dependencies

```bash
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

## ▶️ Running the Project

### Start Backend

```bash
node server.js
```

Backend runs on:

```
http://localhost:5000
```

### Start Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

or

```
http://localhost:5174
```
